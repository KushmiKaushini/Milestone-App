Personal Milestone Tracker

**Momentum** is a sleek, "production-ready" utility app built with Flutter. It helps users track time elapsed since significant life events, focusing on a high-contrast, commercial-grade aesthetic.

Key Features
* **Persistent Storage:** Uses `shared_preferences` to ensure your data stays on the device even after a restart.
* **Modern UI:** Implements a dark-mode gradient aesthetic with "Glassmorphism" design principles.
* **Native Date Picker:** Leverages Flutter's native calendar integration for seamless user input.
* **Optimized Logic:** Real-time date arithmetic using Dart's `DateTime` class.

Tech Stack
* **Framework:** Flutter (3.x+)
* **Language:** Dart
* **Typography:** Google Fonts (Oswald & Lato)
* **Persistence:** Shared Preferences
* **UI Pattern:** StatefulWidget for real-time UI updates
