# Milestone Pro

Milestone Pro is a clean, efficient mobile application built with **Flutter** and **Dart** designed to help users manage their goals and deadlines with precision. 

#  Key Features
* **Deadline Tracking:** Add, edit, and manage milestones with specific target dates.
* **Smart Notifications:** Automated reminders 2 days before a deadline and at the exact time of the event.
* **Snooze Functionality:** One-tap snooze directly from the notification tray.
* **Timezone Aware:** Optimized for the `Asia/Colombo` timezone.
* **Category Management:** Organize tasks by categories like Work, Personal, or Education.

### Tech Stack
* **Frontend:** Flutter (Dart)
* **State Management:** StatefulWidget & setState (optimized for performance)
* **Local Storage:** SharedPreferences
* **Notifications:** flutter_local_notifications
* **Timezone Handling:** timezone (tz.local)
